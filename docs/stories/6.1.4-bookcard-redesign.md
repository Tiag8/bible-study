---

## CRITERIOS DE ACEITACAO

### A. Visual Base do Card

```gherkin
Scenario: Card com estetica parchment
  Given um BookCard na grid
  When observo o visual
  Then o fundo e cream (#F5F0E8)
  And a borda e linen (#EDE8E0)
  And os cantos sao arredondados (12px / rounded-xl)
  And o padding interno e 24px (p-6)
```

**Tarefas Tecnicas**:
- [ ] Background: `bg-white` → cream (#F5F0E8)
- [ ] Border: `border-gray-200/100` → linen (#EDE8E0)
- [ ] Padding: `p-5` → `p-6` (mais generoso)

### B. Hover e Interacao

```gherkin
Scenario: Hover premium no card
  Given um BookCard
  When passo o mouse sobre ele
  Then o card sobe 2px (translateY -2px)
  And a sombra aumenta para warm shadow-md
  And o fundo muda para warm-white (#FEFCF8)
  And a transicao dura 200ms ease
```

**Tarefas Tecnicas**:
- [ ] Hover: `hover:shadow-md` → translateY(-2px) + warm shadow-md
- [ ] Hover bg: warm-white
- [ ] Transition: `transition-all duration-200 ease-out`
- [ ] Manter `cursor-pointer`

### C. Tipografia do Card

```gherkin
Scenario: Hierarquia tipografica no card
  Given um BookCard com dados
  When observo os textos
  Then o nome do livro usa Lora SemiBold em cor espresso (#3C2415)
  And "X capitulos" usa Inter Regular em cor stone (#7A6F64)
  And "X de Y estudados" usa Inter Regular em cor walnut (#5C4033)
  And a porcentagem usa Inter SemiBold em cor espresso
  And "Atualizado ha X" usa Inter Regular em cor sand (#A69B8D)
```

**Tarefas Tecnicas**:
- [ ] Book name: adicionar `font-lora font-semibold` + cor espresso
- [ ] Chapter count: Inter + cor stone
- [ ] Progress text: cor walnut
- [ ] Percentage: cor espresso, font-semibold
- [ ] Last update: cor sand

### D. Progress Bar

```gherkin
Scenario: Progress bar elegante
  Given um BookCard com estudos
  When observo a barra de progresso
  Then o background da track e linen (#EDE8E0)
  And a barra preenchida tem gradiente amber (#B8860B → #8B6508)
  And quando 100% a barra usa success pastel (#4A6741)
  And a barra tem cantos arredondados
```

**Tarefas Tecnicas**:
- [ ] Track: `bg-gray-100` → linen
- [ ] Fill: `bg-blue-600` → gradient amber
- [ ] Complete (100%): cor success pastel verde
- [ ] Zero (0%): tom neutro mais suave

### E. Icone e Badge

```gherkin
Scenario: Icone BookOpen e badge AT/NT
  Given um BookCard
  When observo o header
  Then o icone BookOpen tem fundo amber-light e cor amber (se tem estudos)
  And o icone tem fundo neutro e cor sand (se nao tem estudos)
  And o badge AT/NT usa estilo quente (nao blue/secondary padrao)
```

**Tarefas Tecnicas**:
- [ ] Icon bg (has studies): amber-light (#F5E6C8)
- [ ] Icon color (has studies): amber (#B8860B)
- [ ] Icon bg (no studies): parchment
- [ ] Icon color (no studies): sand
- [ ] Badge AT: tom quente (amber outline)
- [ ] Badge NT: tom quente (amber filled sutil)

### F. Tags

```gherkin
Scenario: Tags em pasteis quentes
  Given um BookCard com tags
  When observo os badges de tag
  Then as tags usam cores pasteis quentes (nao outline cinza)
  And o texto das tags e legivel (contraste adequado)
```

**Tarefas Tecnicas**:
- [ ] Tag badges: aplicar cores pasteis do TAG_COLORS atualizado
- [ ] Texto tags: cor escura adequada para contraste

### G. Cards Sem Estudo

```gherkin
Scenario: Card sem estudos e mais sutil
  Given um BookCard sem nenhum estudo
  When observo
  Then o card tem opacity 0.7
  And o background e levemente mais neutro
  And o hover ainda funciona (incentiva comecar)
```
