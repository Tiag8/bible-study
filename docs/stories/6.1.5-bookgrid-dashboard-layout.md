## CRITERIOS DE ACEITACAO

### A. Background Principal

```gherkin
Scenario: Dashboard com fundo parchment
  Given o DashboardClient carrega
  When observo o layout
  Then o fundo principal e parchment (#FAF6F0) nao cinza
  And o loading state usa cor amber (nao blue)
  And o texto de loading usa cor stone
```

**Tarefas Tecnicas**:
- [ ] `bg-gray-50` → bg parchment (#FAF6F0) no container principal
- [ ] Loading spinner: cor amber
- [ ] Loading text: cor stone

### B. Header da Pagina

```gherkin
Scenario: Titulo com tipografia premium
  Given a pagina de livros (sem livro selecionado)
  When observo o header
  Then "Biblioteca Biblica" usa Lora Bold em cor espresso (#3C2415)
  And o subtitulo usa Inter Regular em cor stone (#7A6F64)
  And o espacamento do header e generoso (mb-8)
```

**Tarefas Tecnicas**:
- [ ] h1: adicionar `font-lora font-bold` + cor espresso
- [ ] Subtitulo: cor stone (era neutral.text.secondary)
- [ ] Margin bottom: `mb-6` → `mb-8`

### C. Secoes AT/NT Estilizadas

```gherkin
Scenario: Section headers com estilo parchment
  Given as secoes Antigo Testamento e Novo Testamento
  When observo os headers
  Then o titulo usa Lora Bold em cor espresso
  And existe uma linha decorativa amber-light ao lado do titulo
  And o contador "(X livros)" usa Inter em cor sand
  And o dot decorativo usa cor amber (AT) ou accent (NT)
  And o espaco entre secoes AT e NT e 48px
```

**Tarefas Tecnicas**:
- [ ] Section h2: `font-lora font-bold` + cor espresso
- [ ] Decorator: linha horizontal amber-light (border-bottom ou pseudo-element)
- [ ] Contador: cor sand
- [ ] Dot AT: cor stone, dot NT: cor amber
- [ ] Space entre secoes: `space-y-8` → gap maior (48px)

### D. Stats do Grid

```gherkin
Scenario: Stats com estilo quente
  Given as estatisticas no topo do grid
  When observo
  Then "X de Y livros" e "X capitulos estudados" usam cor stone
  And os numeros em negrito usam cor espresso
  And o separador | usa cor linen
```

**Tarefas Tecnicas**:
- [ ] Stats text: cor stone
- [ ] Stats bold: cor espresso
- [ ] Separador: cor linen

### E. Grid Responsivo

```gherkin
Scenario: Grid com espacamento generoso
  Given os BookCards no grid
  When observo
  Then o gap entre cards e 20px (era 16px)
  And o padding lateral do conteudo e 32px minimo
  And o grid mantém responsive breakpoints (1/2/3/4 colunas)
```

**Tarefas Tecnicas**:
- [ ] Gap: `gap-4` → `gap-5` (20px)
- [ ] Main padding: `p-6` → `p-8` (32px)
- [ ] Manter breakpoints sm/lg/xl

### F. Empty State

```gherkin
Scenario: Empty state com estilo quente
  Given nenhum livro encontrado (busca sem resultado)
  When observo
  Then a mensagem usa cor stone
  And o visual e quente (nao cinza frio)
```
