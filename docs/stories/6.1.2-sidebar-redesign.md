## CRITERIOS DE ACEITACAO

### A. Visual da Sidebar

```gherkin
Scenario: Sidebar com estetica parchment
  Given a sidebar esta visivel
  When observo o visual
  Then o fundo e ivory (#FFFEF9)
  And a borda direita e linen (#EDE8E0)
  And o logo "Bible Graph" usa Lora Bold em cor espresso (#3C2415)
  And o icone BookOpen usa cor amber (#B8860B)
```

**Tarefas Tecnicas**:
- [ ] Background: `bg-white` → cor ivory customizada
- [ ] Border-right: `border-gray-200` → cor linen
- [ ] Logo text: Adicionar `font-lora font-bold` + cor espresso
- [ ] Logo icon: Cor amber ao inves de primary blue

### B. Navegacao Refinada

```gherkin
Scenario: Nav items com estilo quente
  Given os items de navegacao (Inicio, Grafo, Configuracoes)
  When observo e interajo
  Then o texto usa Inter com cor earth (#6B5B4F)
  And o hover tem background cream (#F5F0E8)
  And a transicao e suave (200ms ease)
```

**Tarefas Tecnicas**:
- [ ] Nav items text: `text-gray-700` → cor walnut/earth
- [ ] Hover: `hover:bg-gray-100` → hover bg-cream
- [ ] Transicao ja existe (transition-colors), manter

### C. Perfil do Usuario

```gherkin
Scenario: Avatar e perfil com estilo amber
  Given o usuario esta logado
  When observo a secao de perfil na sidebar
  Then o avatar tem gradiente amber (nao blue)
  And o nome usa cor espresso
  And o badge de role usa tom pastel quente
  And o separador e linen
```

**Tarefas Tecnicas**:
- [ ] Avatar: `bg-blue-600` → gradiente ou solid amber
- [ ] Nome: cor espresso
- [ ] Borders: `border-gray-100` → cor linen
- [ ] Badge role: ajustar para tom quente

### D. Botao Sair

```gherkin
Scenario: Botao sair com tom pastel
  Given o botao sair na sidebar
  When observo
  Then usa tom pastel danger quente (nao vermelho frio)
  And hover com fundo pastel sutil
```

**Tarefas Tecnicas**:
- [ ] Cor texto: `text-red-600` → tom terracota pastel (#DEC0B8 text mais escuro)
- [ ] Hover: background pastel sutil
